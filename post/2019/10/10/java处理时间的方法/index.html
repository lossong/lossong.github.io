// 创建一个Date对象,采用当前系统时间作为日期
Date date = new Date();
//Thu May 23 01:00:32 CST 2019
System.out.println(date.toString());
//得到年份，获取到的年份需要+1900才是正确的年份
int year = date .getYear() + 1900;
//得到月份，月份和星期都是从0开始计起的
int month = date .getMonth() + 1;
//获得日期
int day= date .getDate();
int hour = date.getHours();
int minute = date .getMinutes();
int second = date .getSeconds();
int week = date.getDay();//获取星期
//格式化Date对象
SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd HH:mm:ss");
System.out.println(sdf.format(date));
//MM表示月份，HH表示24小时进制，mm表示分钟
//时间字符串转Date对象
String date_time = "2012-2-15 23:23:10";
Date date1 = sdf.parse(date_time);

对象的域，可以利用get(~)方法获取，也可以利用set(~,值)去更改值，以及add(~,值)对属性进行加减操作。
Calendar.YEAR//年份
Calendar.MONTH//月份，初始仍然为0
Calendar.DATE//日期
Calendar.DAY_OF_MONTH//日期，和上面的字段意义相同
Calendar.HOUR//12小时制的小时
Calendar.HOUR_OF_DAY//24小时制的小时
Calendar.MINUTE//分钟
Calendar.SECOND//秒
Calendar.DAY_OF_WEEK//星期几，1代表星期日，2代表星期一。。。
