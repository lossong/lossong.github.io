Git，分布式版本控制系统，代码的管理平台

基本指令

初始化仓库 git init

添加文件 git add <file>/<directory> 如 git add .

提交文件仓库(local) git commit -m "description

克隆仓库到本地

1
2
3

git clone repos

git clone -b branch_name repos

将本地的分支推送到远程仓库里

1
2
3

git push [origin <branch_name>] #默认主分支

git push git@github:

从远程仓库更新代码

1

git pull [origin <branch_name>] #默认主分支

分支管理

创建分支 git branch branch_name

切换分支 git checkout branch_name

上面两者的集合 git checkout -b branch_name

查看当前分支 git branch

1
2
3

# *表示当前分支
*branch_name
master

当我们在自己分支上完成自己的任务时，可以将分支合并到master分支上

1
2

# 首先要切换到master分支
git checkout master
1
2

#用于合并指定分支到当前分支
git merge branch_name

删除分支 git branch -d branch_name

其他命令

显示仓库状态 git status

查看修改内容 git diff <file>

查看历史提交记录 git log

查看提交版本号，同时减少输出信息 git log --pretty=oneline

查看分支合并图 git log --graph

版本回退 git reset --hard <版本号>

记录你的每一次命令 git reflog 可以获取版本号

在Git中，用HEAD表示当前版本,上一个版本就是HEAD^，上上一个版本就是HEAD^^，当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100。
